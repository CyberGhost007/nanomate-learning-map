# Nanomate Learning Map

Interactive team learning map prototype for the Nanomate score, KPI, practice, approval, and release flow.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

For GitHub Pages project URLs, build with the repository base path:

```bash
VITE_BASE_PATH=/nanomate-learning-map/ npm run build
```
